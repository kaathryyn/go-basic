# Learning Go Programming Language

Welcome to the Go Programming Language Learning Project! This repository is structured to help you learn Go (Golang) from the ground up, covering fundamental concepts with practical examples.

## Project Structure

```
go-basic/
├── src/                    # Source code directory
│   ├── 01_basics/         # Basic concepts (variables, types, etc.)
│   ├── 02_control_flow/   # Control flow (if/else, loops, etc.)
│   ├── 03_functions/      # Functions and methods
│   ├── 04_structs/        # Structs and interfaces
│   ├── 05_concurrency/    # Goroutines and channels
│   └── exercises/         # Practice exercises
└── docs/                  # Documentation and learning materials
```

## Getting Started

1. Make sure you have Go installed on your system
2. Clone this repository
3. Navigate through each directory in order
4. Read the accompanying markdown files for explanations
5. Try the code examples
6. Complete the exercises

## Prerequisites

- Go 1.21 or later installed on your system
- Basic understanding of programming concepts
- A code editor (VS Code recommended)

## How to Use This Repository

1. Start with the basics in `01_basics/`
2. Read the markdown documentation in each directory
3. Study and run the example code
4. Try to modify the examples to experiment
5. Complete the exercises to test your understanding

## Topics Covered

- Basic Syntax and Data Types
- Control Flow
- Functions and Methods
- Structs and Interfaces
- Concurrency with Goroutines and Channels
- Error Handling
- And more!

Happy coding!
